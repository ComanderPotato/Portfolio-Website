Project

Assignment Planner (unfinished)

ReactJS JavaScript CSS HTML Figma

About

An application that allows students to plan out their upcoming semester assignment timetable, in an easy-to-use way. The assignment planner keeps track of assignment due dates, necessary notes related to the specific assignment, their achieved grade for the assignment, and an overall grade for the entire subject. The application aims to help students who tend to fall behind due to a lack of organization, as well as to reduce stress when it comes to their grades as it is automatically calculated for them, removing the dreaded “Will I pass?” or “How much do I need?” moment. Currently, the application is unfinished and is using local storage to keep information on the users' device. But plans to implement a backend with Firebase to incorporate Google Authentication are in the works, allowing the planner to connect to and update their Google calendar.

Assignment

Lexical / Syntactic Analyzer

Java

About

Both of these were assignments for the subject “Theory of Computer Science”, which delved deep into the theoretical side of CS, most notedly compilers. The first assignment was to implement a deterministic finite automaton, which takes in a string representing an arithmetic statement and produces an output sequence of tokens that represent the numbers and operators. The second assignment was to implement a parser that checks the syntactic correctness of a simplified subset of the Java programming language and builds a parse tree representing the program.
